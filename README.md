🏠 Interactive 3D Indoor Simulation

This is a simple interactive 3D project built using **C++ and OpenGL**.  
It shows a room where a fan rotates and a cat can walk and jump.  
Users can control everything using the keyboard.

---

🚀 Features

- Ceiling fan rotation  
- Cat walking  
- Cat jumping  
- Day/Night mode  
- Camera rotation  
- Zoom in and out  

---

🎮 Controls

- F → Fan On/Off  
- S → Cat Move On/Off  
- J → Cat Jump  
- L → Day/Night Mode  
- + → Zoom In  
- - → Zoom Out  
- Arrow Keys → Rotate Camera  
- ESC → Exit  

---

🛠️ Tools Used

- C++  
- OpenGL  
- GLUT / FreeGLUT  

---

▶️ How to Run

1. Open the project in Visual Studio / CodeBlocks  
2. Make sure OpenGL and GLUT are installed  
3. Compile and run the code
